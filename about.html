dccode.org/browser

This is a fast, unofficial interface to the Unofficial DC Code released by the DC Council on April 4th, 2013. The Code, previously encumbered by copyrights and terms of service, were made available as a public domain download of Word documents. This makes that download into something useful and browseable.

Unlike the official site, this interface quickly improving and is completely free: you can copy it, improve it, and see how it works.

API

The API is a work in progress. Currently you can request JSON-structured files of each section.

http://dccode.org/browser/sections/3-201.json

In which 3-201 is a code identifier, in this case meaning Title 3, section 201.

Please report any bugs for the browser on GitHub!
